# ROCHA-3D
Rocha 3D é um projeto especializado em soluções completas de impressão 3D, oferecendo criação, ajuste e produção de peças personalizadas. Atuando em um ecossistema criativo comparável a 300 hectares de possibilidades, transformamos ideias em modelos reais com qualidade, precisão e atendimento direto via WhatsApp.
